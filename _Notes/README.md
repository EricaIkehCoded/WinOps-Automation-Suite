# _Notes — Personal Reference Library

This folder contains personal reference files built throughout the
WinOps Automation Suite project. Each file documents commands,
concepts, and code snippets learned during the AZ-040T00 course.

## Files

| File | Description |
|---|---|
| `git-commands-reference.ps1` | Core Git commands used throughout the project |
| `folder-structure-snippet.ps1` | PowerShell snippet used to build the project folder structure |
| `Scratchpad.ps1` | Working notes and earlier iteration history for Phase 1 |

## Purpose

These notes serve two purposes:
- A personal reference to reinforce learning
- A transparent record of the problem solving and iteration
  process behind each phase of the project

  > "Good sysadmins don't just solve problems — they document how."
