## 📌 About This Project

This portfolio project was built as part of the **Microsoft Software and Systems Academy (MSSA)**
Server and Cloud Administration program. It demonstrates hands-on proficiency with Windows
PowerShell automation across 9 skill areas mapped directly to the AZ-040T00 course curriculum.

The fictional company **Contoso Federal Solutions** serves as the scenario backdrop — a
government contractor needing a modular PowerShell toolkit to automate repetitive
administrative tasks across their Windows Server environment.

---

## 🛠️ Skills Demonstrated

- PowerShell cmdlet discovery and syntax
- Pipeline filtering, sorting, and output
- PSProviders and PSDrives
- WMI and CIM system queries
- Variables, arrays, hash tables, and scripting logic
- PowerShell remoting and persistent sessions
- Azure resource management with PowerShell
- Microsoft 365 administration with PowerShell
- Background jobs and scheduled tasks

## 🚀 How to Use This Repo

Each phase folder contains a standalone PowerShell script with inline comments
explaining what it does and why. Scripts are designed to be readable and
educational — not just functional.

**Requirements:**
- PowerShell 7.4 or later
- Windows Server or Windows 10/11 environment (for Windows-specific cmdlets)
- Az PowerShell module (Phase 7)
- Microsoft 365 PowerShell modules (Phase 8)
